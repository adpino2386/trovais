What is the torque spec for the CNC-7 main drive shaft?
CNC-7 Main Drive Shaft — Torque Specification

According to the SINUMERIK 828D maintenance manual, the maximum torque for the main drive shaft is 420 Nm at 3,200 RPM. Exceeding this value without active load monitoring will trigger emergency stop protocol E-04.

Quelles sont les étapes de maintenance pour la pompe hydraulique P-240?
Maintenance — Pompe hydraulique P-240

Selon le manuel de maintenance (rév. 2023), la pompe P-240 requiert une inspection toutes les 500 heures d'opération. Les étapes: (1) vérifier le niveau d'huile, (2) inspecter les joints pour fuites, (3) tester la pression de sortie (min. 280 bar), (4) nettoyer le filtre à particules.
📄 manuel_P240_rev2023.pdf — p. 47 📄 sop_maintenance_hydraulique.docx — p. 12
